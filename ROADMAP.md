# Roadmap

This roadmap keeps small maintenance work visible for contributors.

## Current focus

- Keep command catalog metadata and policy target coverage aligned.
- Improve examples for dashboard and bot runtime integrations.
- Add tests for serialization compatibility before changing public helpers.
- Keep the security baseline and CI green on every pull request.

## Good first issues

- Add README examples for common moderation-only policy setups.
- Add tests for channel allow-list and role allow-list combinations.
- Document the expected dashboard JSON shape with a copy-paste fixture.
- Review command summaries for clarity and consistency.

## Later

- Add generated markdown tables for command categories and policy targets.
- Add package publishing automation once the package name and release channel are stable.
- Add compatibility examples for popular Discord bot frameworks.
